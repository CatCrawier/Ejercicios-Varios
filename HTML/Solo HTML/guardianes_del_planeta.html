Situación General

La naturaleza enfrenta una presión creciente por la pérdida de hábitats, la contaminación y el cambio climático.

Datos relevantes

La conservación es urgente. Es importante proteger bosques y fauna.

Según expertos, cada ecosistema perdido debilita el equilibrio global.

La biodiversidad sostiene nuestra alimentación, el agua, el suelo y la salud del planeta.

Lista Roja de la UICN

Sitio oficial que muestra el estado de conservación de especies de animales, plantas y hongos, indicando qué tan cerca están de la extinción. La información se basa en criterios científicos rigurosos y es una referencia clave para la conservación global. (La información se actualiza periódicamente.)

Ejemplos de acciones

  • Reciclar y reducir plásticos.
  • Apoyar reservas naturales.
  • Promover educación ambiental.

Pasos para colaborar

  1. Informarse sobre especies locales.
  2. Participar en campañas de restauración.
  3. Difundir prácticas sostenibles.

¿Sabías que...?

Muchas especies polinizadoras están en declive.

Proteger la biodiversidad también protege nuestra salud.

Glosario de Informática (A-Z)

A

  • Algoritmo: conjunto de pasos para resolver un problema.
  • API: interfaz para comunicar aplicaciones.
  • Autenticación: proceso para verificar identidad.

B

  • Backup: copia de seguridad de datos.
  • Bit: unidad mínima de información (0 o 1).
  • Bug: error en software o hardware.

C

  • Cache: memoria rápida para acelerar accesos.
  • CPU: unidad central de procesamiento.
  • Cifrado: protección de datos mediante codificación.

D

  • DNS: traduce dominios a direcciones IP.
  • Depuración: proceso de localizar y corregir errores.
  • Data center: centro físico con servidores y red.

E

  • Encriptación: técnica para proteger información.
  • Ethernet: tecnología de red local cableada.
  • Endpoint: punto final de comunicación en una red o API.

F

  • Firewall: sistema que filtra tráfico de red.
  • Firmware: software interno de un dispositivo.
  • FTP: protocolo para transferir archivos.

G

  • GPU: procesador especializado en gráficos.
  • Git: sistema de control de versiones.
  • Gateway: puerta de enlace entre redes.

H

  • HTML: lenguaje de marcado para páginas web.
  • HTTP: protocolo de transferencia web.
  • Hosting: servicio de alojamiento de sitios o apps.

I

  • IP: dirección de identificación en red.
  • IoT: dispositivos conectados a internet.
  • IDE: entorno de desarrollo integrado.

J

  • Java: lenguaje de programación multiplataforma.
  • JavaScript: lenguaje de programación para web.
  • JSON: formato de intercambio de datos.

K

  • Kernel: núcleo del sistema operativo.
  • Kubernetes: plataforma para orquestar contenedores.
  • Keylogger: software que registra pulsaciones de teclado.

L

  • LAN: red de área local.
  • Linux: familia de sistemas operativos de código abierto.
  • Latencia: tiempo de demora en una comunicación.

M

  • Machine Learning: rama de IA basada en datos.
  • Malware: software malicioso.
  • Metadatos: datos que describen otros datos.

N

  • Nube (Cloud): servicios de cómputo por internet.
  • Nodo: dispositivo o punto dentro de una red.
  • NAT: traducción de direcciones de red.

Ñ

  • Ñ en Unicode: carácter representado como U+00D1 (mayúscula) y U+00F1 (minúscula).
  • Ñ en UTF-8: carácter codificado en 2 bytes dentro de este estándar.
  • Ñ en ASCII extendido: carácter soportado en tablas de códigos como ISO-8859-1 y Windows-1252.

O

  • Open Source: software con código fuente abierto.
  • OCR: reconocimiento óptico de caracteres.
  • Overclocking: aumento de frecuencia del hardware.

P

  • Protocolo: reglas para comunicación en red.
  • PHP: lenguaje de scripting para servidor web.
  • Ping: prueba de conectividad y latencia.

Q

  • Query: consulta de datos en una base.
  • QoS: calidad de servicio en redes.
  • Queue: estructura de datos tipo cola (FIFO).

R

  • RAM: memoria principal de acceso aleatorio.
  • Router: equipo que enruta tráfico entre redes.
  • Repositorio: almacenamiento de código y versiones.

S

  • Servidor: sistema que ofrece servicios en red.
  • SQL: lenguaje para bases de datos relacionales.
  • SSL/TLS: protocolos de seguridad para cifrado.

T

  • TCP/IP: conjunto base de protocolos de internet.
  • Token: unidad de autenticación o seguridad.
  • Thread: hilo de ejecución dentro de un proceso.

U

  • URL: dirección de un recurso en la web.
  • USB: estándar de conexión de periféricos.
  • UX: experiencia de usuario en productos digitales.

V

  • VPN: red privada virtual.
  • Variable: espacio para almacenar un valor.
  • Virtualización: creación de recursos virtuales.

W

  • WAN: red de área amplia.
  • WebSocket: comunicación bidireccional en tiempo real.
  • Wi-Fi: tecnología de red inalámbrica.

X

  • XML: lenguaje de marcado extensible.
  • XPath: lenguaje para consultar nodos XML.
  • XSS: ataque de inyección de scripts.

Y

  • YAML: formato de serialización de datos.
  • Yottabyte: unidad de almacenamiento muy grande.
  • Y2K: problema informático del cambio de milenio.

Z

  • Zero-day: vulnerabilidad sin parche disponible.
  • ZIP: formato de compresión de archivos.
  • Zettabyte: unidad de almacenamiento masiva.
